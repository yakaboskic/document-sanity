\section{Example: Using Variable Templates}

% ============================================================
% BASIC USAGE
% ============================================================

% Simple variable replacement (no formatting)
The standard error for OTG GWAS was {{OTG_GWAS_SE}}.

The standard error for PIGEAN GWAS was {{PIGEAN_GWAS_SE}}.

% ============================================================
% FORMATTED NUMERIC VALUES
% ============================================================

% Decimal places with :.Nf (where N is number of decimals)
The R² value was {{PIGEAN_RIDGE_REGRESSION_R2:.2f}} with
an adjusted R² of {{PIGEAN_RIDGE_REGRESSION_R2_ADJ:.2f}}.

% Scientific notation with :.Ne (where N is number of decimals)
The regression showed strong significance
(p = {{PIGEAN_DIRECT_REGRESSION_P_VALUE:.2e}},
coefficient = {{PIGEAN_DIRECT_REGRESSION_COEF:.2f}}).

% Another example with very small p-values
The indirect support model was highly significant
(R² = {{PIGEAN_INDIRECT_REGRESSION_R2:.2f}},
p = {{PIGEAN_INDIRECT_REGRESSION_P_VALUE:.2e}}).

% ============================================================
% THOUSANDS SEPARATORS
% ============================================================

% Use :, for thousands separator (works with pre-formatted strings)
We analyzed a total of {{NUM_PIGEAN_TRAITS}} traits across
{{NUM_UNIQUE_DRUG_PROGRAMS}} drug programs.

The dataset included {{NUM_PORTAL_PIGEAN_TRAITS}} traits from the portal,
{{NUM_ORPHANET_PIGEAN_TRAITS}} from Orphanet, and
{{NUM_OMIM_PIGEAN_TRAITS}} from OMIM.

% ============================================================
% CONFIDENCE INTERVALS
% ============================================================

% Multiple variables in one sentence with formatting
The conditional categorical adaptation test showed an exact estimate of
{{CCAT_EXACT_PIGEAN_ALL_ALL_AREAS:.2f}}
(95\% CI: {{CCAT_CI_LOWER_PIGEAN_ALL_ALL_AREAS:.2f}}--{{CCAT_CI_UPPER_PIGEAN_ALL_ALL_AREAS:.2f}}).

% ============================================================
% UNDEFINED VARIABLES (PLACEHOLDER WORKFLOW)
% ============================================================

% These variables don't exist in the JSON yet - they will show as XXXX
% This is useful when writing the manuscript before running analysis

The new validation metric showed {{NEW_VALIDATION_METRIC:.3f}}.

We observed an improvement of {{IMPROVEMENT_PERCENTAGE:.1f}}\% over baseline.

The exploratory analysis revealed p = {{EXPLORATORY_P_VALUE:.2e}}.

% ============================================================
% COMPLEX EXAMPLE
% ============================================================

% Combining multiple variables in a paragraph
Our analysis of {{NUM_UNIQUE_MAPPED_TRAITS}} unique mapped traits
across {{NUM_UNIQUE_DRUG_TARGETS}} drug targets revealed that
the combined regression model achieved an R² of {{PIGEAN_COMBINED_REGRESSION_R2:.2f}}
(p = {{PIGEAN_COMBINED_REGRESSION_P_VALUE:.4f}},
coefficient = {{PIGEAN_COMBINED_REGRESSION_COEF:.2f}}).
