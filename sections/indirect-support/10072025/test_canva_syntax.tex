\section{Test Canva Syntax}

This section demonstrates the new {{canva:page}} syntax for Canva figures.

\subsection{Figure from Canva Page 1}

\begin{figure}[ht!]
    \centering
    {{canva:1}}
    \caption{Test figure from Canva page 1}
    \label{fig:canva-test-1}
\end{figure}

\subsection{Figure from Canva Page 2}

\begin{figure}[ht!]
    \centering
    {{canva:2}}
    \caption{Test figure from Canva page 2 with custom width}
    \label{fig:canva-test-2}
\end{figure}

This syntax automatically resolves to the correct figure path based on the manifest configuration.
